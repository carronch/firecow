The Zero-Ban Provisioning Flow

1

Human Input (The Spreadsheet UI)

Our agency team safely inputs the client's raw data into the custom React Admin Dashboard. This requires precisely three steps to provision a brand new site:

A. Add Supplier

Input Company Name, WhatsApp, and the external Calendar API url (FareHarbor/Checkfront).

B. Add Tours

Attach specific tours to the supplier. Set the price, descriptions, and securely upload Hero images to Cloudflare R2.

C. Add Site

Link the Tours to the Site entity, assign a globally unique slug, and define the Cloudflare Pages deploy hook.

Once these three database schemas are fulfilled, the Worker API safely sanitizes and commits the payload to the centralized D1 Edge Database.

2

Trigger Cloudflare Build Hook

The team clicks "Deploy" in the Admin Dashboard. This instantly POSTs to the unique Cloudflare Pages `deploy_hook` assigned exactly to that client's target URL. No human bot-like crawling.

3

Astro SSR Template Compilation

Cloudflare provisions our singular Astro Monorepo. Astro executes server-side (SSR) and dynamically queries the D1 Database for ONLY that single client's parameters without loading massive headless CMS bloat.

4

SEO Schema Generation

During the split-second DOM render, Astro constructs a perfect `JSON-LD TouristAttraction` schema mapping the client's catalog, embedding it strictly into the HTML ``. This guarantees Google parses the deeply nested catalog perfectly on Day 1.

5

Global Cron Job Registration

The site goes live across Cloudflare's 300+ data centers, and the backend Cron Automatically adds this new client to its 30-minute sync rotation to guarantee FareHarbor capacity matching.

The Google Algorithm Strategy

Interacting with Google's ecosystem at scale is a minefield of algorithmic bot bans. We have engineered a strict Hybrid-Automation model to entirely bypass these penalties while still achieving enterprise expansion.

Google Business Profile (GBP) Verification

Instead of using APIs to auto-create profiles (which triggers immediate spam-suspensions), our system generates a highly-trusted "GBP Launch Kit" for precise human copy/pasting. This guarantees instant algorithmic verification by stacking massive domain authority signals:

  • Domain Identity (Resend): We provision a strict `info@clientURL.com` inbox via Resend. Google's algorithm sees the email precisely matches the application URL.
  • Local Authority (Twilio): We provision a geographic virtual phone number via Twilio, routing any Google verification texts directly into our Crisp Unified Inbox.
  • 2025 Video Verification Prep: Because Google now mandates live unedited video tests for Service Area Businesses, our platform automatically generates a custom "What to Film" script for the client (branded vehicles, tool access, local landmarks) ensuring a first-try pass.

Google Ads API via MCC

Creating brand new ad accounts repeatedly triggers billing lockouts. Instead, our automation connects strictly to an aged, trusted Agency Manager Account (MCC). High intent campaigns are safely spawned instantly under our MCC umbrella via API.
